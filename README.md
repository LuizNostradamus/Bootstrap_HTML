# Bootstrap_HTML

Repositório de estudo com exemplos e exercícios simples usando HTML e Bootstrap, criado para fins de aprendizado.

## Sobre
Este repositório contém pages e snippets em HTML estilizados com Bootstrap para praticar layout responsivo, componentes e boas práticas de front-end. É indicado para iniciantes que querem aprender como estruturar páginas usando o framework Bootstrap.

## Objetivo de aprendizado
- Entender a estrutura básica de um documento HTML.
- Aprender a integrar o Bootstrap (CDN e arquivos locais).
- Praticar o sistema de grid e layout responsivo.
- Usar componentes do Bootstrap: Navbar, Cards, Forms, Modals, Buttons.
- Customizar estilos com CSS adicional.

## Estrutura do repositório
- `index.html` — exemplo principal com layout básico.
- `pages/` — exemplos de páginas (ex.: `form.html`, `grid.html`, `cards.html`).
- `assets/`
  - `css/` — estilos personalizados.
  - `js/` — scripts de exemplo.
  - `img/` — imagens usadas nos exemplos.
- `README.md` — este arquivo.
